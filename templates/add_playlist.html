Как добавить плейлист?

  1. Откройте Spotify (веб-версия или приложение)
  2. Найдите нужный плейлист
  3. Нажмите на три точки (...) рядом с названием плейлиста
  4. Выберите "Поделиться" → "Скопировать ссылку на плейлист"
  5. Вставьте ссылку в поле выше

  1. Откройте Apple Music
  2. Найдите нужный плейлист
  3. Нажмите на три точки (...) рядом с названием
  4. Выберите "Поделиться плейлистом"
  5. Скопируйте ссылку
  6. Вставьте ссылку в поле выше

Важно: Для мониторинга плейлистов Яндекс.Музыки они должны быть публичными!
  1. Откройте Яндекс.Музыку в браузере
  2. Найдите нужный плейлист
  3. Убедитесь, что плейлист публичный (доступен по ссылке)
  4. Скопируйте ссылку из адресной строки браузера
  5. Вставьте ссылку в поле выше

Пример ссылки: https://music.yandex.ru/playlists/1766199b-8355-0677-923c-c65e22382e2f
